INTRODUCTION
Ce projet m'a permis de réaliser une analyse des données de ventes d'une librairie en ligne fictive.
J'ai tout d'abord nettoyé et réorganisé les données puis je les ai analysées.
Enfin, j'ai testé 5 corrélations entre l'âge des clients, leurs genres et d'autres éléments comme le panier moyen ou la catégorie des produits achetés.

ORGANISATION
3 dossiers :
- Sources : contient les données initiales du projet.
- Export : contient les données cleansées.
- Graphiques : contient l'ensemble des graphiques du projet créés pour ce projet

3 Jupyter Notebooks :
- Nettoyage.ipynb : récupère les données du dossier Sources, les nettoie, et enregistre le résultat dans le dossier Export.
- Analyse : récupère les données du dossier Export, les analyse et produit des graphiques enregistrés dans le dossier Graphiques.
- Corrélations : récupère les données du dossier Export, test les corrélations et produit des graphiques enregistrés dans le dossier Graphiques.
